# Projeto: Calculadora de Salário Final (COBOL)

Este programa foi desenvolvido como parte do **Projeto 3 da Semana 5 do curso Acelerador**, com o objetivo de colocar em prática os conhecimentos de **COBOL**, utilizando **parágrafos** e o comando **PERFORM** para organizar a lógica do programa.

## Funcionalidades

O sistema recebe os dados do funcionário e calcula o **salário final** com base no seu **tempo de empresa**.

### Regras de bônus

As seguintes regras de bônus são aplicadas:

- **Até 1 ano de empresa:** 5% de bônus.
- **De 2 a 5 anos de empresa:** 10% de bônus.
- **Acima de 5 anos de empresa:** 15% de bônus.

## Objetivos do Projeto

- Praticar a criação de programas em COBOL.
- Utilizar parágrafos para organizar o código.
- Aplicar o comando **PERFORM** para controlar o fluxo de execução.
- Realizar cálculos condicionais com base em regras de negócio.
- Exibir o salário final já acrescido do bônus correspondente.
